ITEC 1018: Linux Essentials

This course went over the Linux operating system. It went over many different commands in the command line to pull up the different directories, files, moving files, and even looking at the computer system information. It also included things such as hidden files, compressing files, deleting files, and making sure that there is a prompt before deleting files.

This course was very interesting and gave a broad overview of the Linux system. I definitely had a good time going over it all and learning the details. I can’t say that I can fully navigate every single thing of Linux, but I did learn the basics of navigating and find that I do really enjoy the operating system. It’s a system that I plan to explore further and possibly even begin to use on my own.
